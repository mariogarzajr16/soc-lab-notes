# SOC Lab Notes

This repository contains hands-on security operations lab work focused on SOC Tier 1 analyst skills.

The work here includes:
- Alert triage and investigation
- Log analysis using Splunk
- Network traffic analysis with Wireshark
- Vulnerability scanning and reconnaissance with Nmap and Kali Linux
- Identification of indicators of compromise (IOCs)
- Documentation of findings and remediation steps

All content is based on lab environments, simulated data, and personal practice.
