# Introduction to Splunk Enterprise Security (ES)

## Overview
This document summarizes hands-on exposure to Splunk Enterprise Security concepts through structured training.

## SOC Capabilities Observed
- Centralized security monitoring
- Alerting and notable events
- Analyst-driven triage workflows (Analyst Queue)
- Integration with incident response processes (SOAR)

## Analyst Perspective
Splunk ES provides a consolidated view of security events, allowing analysts to quickly prioritize alerts, investigate activity, and escalate incidents when appropriate (Threat Topology).
